Quickstart for dApp Developers

This guide will walk you through how to implement the basics of ENS into your dApp. This quickstart guide assumes you are building with react

Before you can look up on-chain ENS information, you will need to pick an RPC provider to interface with Ethereum. Read more about picking an RPC provider at RPC.

Choose your client

Before making your first request, you need to pick which library you will use. In the following example, you can see how to install each library.

# Run this in your terminal
npm install wagmi ethers
Check out our list of first-party Libraries
