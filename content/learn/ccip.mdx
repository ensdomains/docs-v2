# CCIP

<WIP />

[EIP-3668](https://eips.ethereum.org/EIPS/eip-3668), otherwise known as CCIP Read, is an Ethereum standard that enables any smart contract to load data from an external source.
This allows us to load information about names and subnames, from an L2, Database, or a regular REST endpoint.
